db_set(Db *dbp, DbEnv *envp, InputIterator first,
    InputIterator last)

Iteration constructor.

Iterates between first and last, setting a copy of each of the sequence of elements as the content of the container object. Create a std::set/hash_set equivalent associative container. Insert a range of elements into the database. The range is [first, last), which contains elements that can be converted to type ddt automatically. This function supports auto commit. See the handle requirement in class details to pass correct database/environment handles.

Parameters

dbp

The database handle.

envp

The database environment handle.

last

The open boundary of the range.

first

The closed boundary of the range.

db_set(const self &x)

Copy constructor.

Create an database and insert all key/data pairs in x into this container. x's data members are not copied. This function supports auto commit.
