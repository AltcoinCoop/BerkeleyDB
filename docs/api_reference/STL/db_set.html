Chapter 7.  The db_set Class

This class is the combination of std::set and hash_set.

By setting database handles of DB_BTREE or DB_HASH type, you will be using an equivalent of std::set or hash_set. This container stores the key in the key element of a key/data pair in underlying database, and keeps the data element empty; Database and environment handle requirement: The same as that of db_map .
