ElementRef

Function Details

ElementRef(iterator_type *pitr=NULL)

Constructor.

This constructor can optionally accept an iterator to associate its wrapped element with the underlying key/data pair in the database. If no iterator is specified, this object is only a wrapper and can't access the database.

Parameters

pitr

The iterator owning this object. 

ElementRef(const ddt &dt)

Constructor.

This constructor initializes an ElementRef wrapper without an iterator, thus it can only be used to wrap data element in memory, can't access database.

Parameters

dt

The base class object to initialize this object. 

ElementRef(const self &other)

Copy constructor.

This constructor initializes an ElementRef instance from an existing one, copying its wrapped element, and if the "other" object has a valid iterator, the constructor will duplicate the iterator and thus the underlying cursor, so that the new instance can access the database.

Parameters

other

The object to clone from.
